# dotfiles

#### Up and running mostly use maintained repos:

 * zsh
 * `chsh -s $(which zsh)`
 * vim
 * tmux
 * oh-my-zsh (https://github.com/robbyrussell/oh-my-zsh)
 * yadr (https://github.com/skwp/dotfiles)

```shell
 cd $HOME
 git clone clone https://github.com/KlotzAndrew/dotfiles.git
 cd $HOME/dotfiles
 rake install
```
